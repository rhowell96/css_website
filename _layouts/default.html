The Future

After using float: right or left, you might find that the website struggles to figure out where exactly the floating should end. In this case, you can use clear: right, clear: left, or clear: both to end the float instruction to your computer and make the next paragraph begin below the floating object.

overflow: hidden will achieve the same effect, and it's recommended that you pop an overflow command into entire wrappers (e.g. biowrappers) to ensure that all text within their bounds will cleanly break from the previous element. Keep in mind that this will cut off dropdown menus if used irresponsibly.

Background

Basically, overflow: hidden will hide all of the content of an element that spills out from a container of a set height. overflow: visible will allow the content that would have otherwise been cut off to flow freely across the page, regardless of height restrictions. overflow: scroll will allow the user to scroll around in a box of the defined dimensions.
